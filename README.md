# Automation Projects Repository — TwinCAT & TIA Portal

This repository consolidates **industrial automation projects** developed using **Beckhoff TwinCAT 3** and **Siemens TIA Portal**.  
It provides a structured workspace for PLC and HMI projects, reusable control modules, and deployment utilities used in industrial automation systems.

---

## Overview

The repository includes four main automation projects:

1. **Dual Pump Water Supply Control System**  
   - Go to: [`/water_supply_system/README.md`](projects/dual_pump_water_supply/README.md)

2. **Identification Station**  
   - Go to: [`/identification_station/README.md`](projects/identification_station/README.md)

3. **Conveyor Station**  
   - Go to: [`/conveyor_station/README.md`](projects/conveyor_station/README.md)

4. **NC Machine**  
   - Go to: [`/nc_machine/README.md`](projects/nc_machine/README.md)

5. **UI for conveyor control**
   - Go to [`/UI_ConveyorControl/README.md`](projects//UI_ConveyorControl/README.md)

Each project contains its own **README.md** file detailing objectives, operation, and control logic.

To download the project
1. Go to: https://github.com/hadao169/automation-projects
2. Click green "Code" button → "Download ZIP"
3. Extract to your workspace
4. Ready to use!
---

## Technology Stack

- **Beckhoff TwinCAT 3** (Build ≥ 4020)  
- **Siemens TIA Portal V18** or newer  
- **Git / GitHub** for version control and collaboration

---

## Configuration

- Adjust hardware mappings and device configurations according to your PLC setup.  
- Update alias paths and I/O assignments to match the target topology before runtime testing.

---

## Usage Guidelines

1. Follow the standardized folder hierarchy when adding new projects.  
2. Maintain consistent variable naming and structured documentation.  
3. Include detailed comments within your logic for traceability.  

> **Note:** Each project is self-contained. Refer to its internal README.md for full operational details.
